# Tension
basically anxiety the game

bright white and blurry environment

two many things happening all at once but only appearing along the edges of your vision

loud overwhelming music that makes you feel nervous

the danger is always coming from where you arent looking

theres a time factor involved so you don't really get a chance to gather your berings
and the game path will be randomly generated so you never play the same game twice
the game path changes and evolves so retracing your steps is also not an option
